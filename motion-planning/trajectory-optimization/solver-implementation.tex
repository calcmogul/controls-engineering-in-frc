\section{Solver implementation}

\textit{Numerical Optimization}, Second Edition by Nocedal and
Wright\footnote{\url{https://www.math.uci.edu/~qnie/Publications/NumericalOptimization.pdf}}
is a useful introductory resource for implementing optimization problem solvers.
Sleipnir provides a list of resources used to implement
itself.\footnote{\url{https://sleipnirgroup.github.io/Sleipnir/md_docs_2resources.html}}
